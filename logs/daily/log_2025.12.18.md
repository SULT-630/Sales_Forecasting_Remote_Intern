# Daily Log — 2025-12-17

## Objective
- Discover any hidden pattern behind total price < base price
- Multi-variate analysis with visualization (筛选特征和解释力度)
- Spatial feature division
- External variable effect and lag-off effect

## Work Done
- For every feature: 信息表达程度，关系形状，方向与稳定性，
    - total price v.s log sales: 整体负相关，分区50，150，300，300+，分段非线性，不能直接使用在线性模型中？每个竖条近似泊松/正态分布，均值在分段中单调,整体上期望销量随着total price上升而下降，但是在total price = 300左右有反常上升; 价格对于销量预测的信息有强解释度，特征应当保留
    - For base price v.s log sales: 整体负相关，均值变换相比total price更平缓，分区近似相同，竖条也近似泊松/正态分布，但是没有在300处反常上升，信息相对于total price较少，但也符合直觉，可以考虑对比模型再决定是否保留特征，或者考虑变换为，base-total或者观察能否用total+discount代替
    - For discount ratio v.s log sales: 整体负相关，除了到1时反常上升，符合业务逻辑，拐点很少，比较干净，说明确实折扣率越高销量越高，对于全商品，价格区间都适配，是强特征，必须保留
    - For is_featured_sku, is_displayed_sku, is_discount_sku v.s log sales: 全部都是为0的时候销量显著更高，代表是强有效特征，mean与median几乎重合
    - For sku_id, store_id v.s log sales: 明显不同的id之间有显著的销量层级，样本量和曲线趋势基本没有相关性，说明分布健康可信；此外在高销量的店铺中，median显著比mean高，说明有稳定的低效量尾部，也就是冷门sku
- Summary： log(sales)=α - SKU基准 + f(price)​​ - 价格效应 + γpromo - 促销 + βstore - 门店偏移 + 噪声
